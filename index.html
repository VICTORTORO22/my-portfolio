Proyectos

APP de frases motivadoras

Aplicación en React que genera frases motivadoras al presionar un botón. La aplicación también cambia el fondo con cada nueva frase.

Aplicación en React que muestra el estado del clima según la ubicación. La aplicación utiliza la geolocalización para permitir la ubicación actual y proporcionar la información.

Rick And Morty

Aplicación en React que consume la API de Rick And Morty para mostrar información sobre los personajes y episodios. La aplicación permite a los usuarios buscar y filtrar información de manera interactiva

Pokedex

Aplicación en React la cual pide al usuario el nombre del entrenador y luego lo lleva a una página principal donde se muestra todos los pokemon disponibles, desde esta pagina, los usuarios pueden filtrar los pokemones por nombre o tipo, al seleccionar el pokemon, la aplicación muestra una pagina con todas las especificaciones del pokemon, incluyendo su tipo, habilidades, estadísticas y movimientos.
